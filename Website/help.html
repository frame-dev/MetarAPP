For what is this Program

This program is used to check the Metar data. Also you can use this for the ILS Frequency and Heading.
Only thing you need to now is the ICAO Code from the Airport. Otherwise you can check it on the APP.


The Wheater Radar is from the Website rainviewer.com

This Program uses the API from api.checkwx.com and the api.flightplandatabase.com.
For this Program is at least Java 1.8 required.
